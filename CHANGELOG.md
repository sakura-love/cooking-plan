# 更新日志

本文件记录 `releases/` 目录中已生成 APK 的版本变化。由于项目当前不是 Git 仓库，早期版本无法逐条追溯提交记录；以下内容按 APK 时间线、现有功能形态和开发记录整理。

## v1.2.8 - 2026-06-19

- 内置菜谱库扩展到 200+ 道，覆盖更多家常菜、家庭可做特色菜、凉菜、小吃和甜品。
- 大量默认菜谱补充了更细的任务式步骤，备菜、腌制、爆香、收汁等操作拆分更清楚。
- 首页新增分类目录和分组浏览，用户不依赖搜索也能快速翻找菜谱。
- 首页搜索升级为支持菜名、分类、简介和食材关键词。
- 默认菜谱改为自动补齐缺失项，未接入 AI 的用户也能直接使用完整离线菜谱库。

## v1.2.4 - 2026-05-24

- 每周计划支持同一餐安排多道菜。
- 选择菜谱后不再自动关闭弹窗，方便连续加菜。
- 已选菜谱在选择弹窗中显示“已选”，避免重复添加。
- 点击早餐、午餐、晚餐可进入新的餐次详情页。
- 餐次详情页展示该顿饭的所有菜谱，支持进入菜谱详情和单独移除。
- 去掉“今天下厨”标签前多余的“今”。
- 首页菜谱卡片去掉与分类标签重复的左侧单字分类块。
- 收藏夹去掉多余的“心 收藏”标签，仅保留爱心按钮。
- 每周计划每日卡片新增“买菜清单”入口。
- 单日买菜清单自动汇总当天计划菜谱所需食材，并支持勾选已买。

## v1.2.3 - 2026-05-24

- 首页“正在做”卡片改为单层普通卡片，去掉多层色块和内框。
- 设置页移除顶部说明卡片，让设置项更直接。
- 调整 API 获取教程位置，使其与 API 设置并列显示，并位于 API 设置下方。

## v1.2.2 - 2026-05-24

## v1.2.1 - 2026-05-24
